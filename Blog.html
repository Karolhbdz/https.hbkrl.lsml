¿Que es Java?

Es un lenguaje de programación orientado a objetos, ampliamente utilizado para desarrollar aplicaciones tanto web como de escritorio, móviles (especialmente en Android) y sistemas empresariales. Fue creado en 1995 por Sun Microsystems (ahora propiedad de Oracle). Java se destaca por su portabilidad, lo que significa que puedes escribir el código una vez y ejecutarlo en cualquier plataforma que tenga la Máquina Virtual de Java (JVM), sin necesidad de cambiarlo.

Sus caracteristicas

Orientado a objetos : Todo en Java se basa en objetos y clases, lo que facilita la organización y reutilización del código.

Multiplataforma : El famoso lema de Java es "escribe una vez, ejecuta en cualquier lugar" (WORA), gracias a que el código Java se ejecuta en la Máquina Virtual de Java (JVM) , que está disponible en Múltiples sistemas operativos.

Seguro : Java tiene mecanismos que permiten controlar el acceso a los recursos del sistema y manejar errores de forma segura, lo que lo convierte en un lenguaje robusto.

Gestiona la memoria automáticamente : Incluye un recolector de basura (Garbage Collector) que se encarga de liberar la memoria que ya no se utiliza, evitando fugas de memoria.

Multihilo : Java soporta la ejecución de múltiples procesos de forma concurrente, lo que ayuda a aprovechar mejor los recursos del procesador.

Bibliotecas extensas : Java cuenta con una gran cantidad de bibliotecas estándar y frameworks que facilitan el desarrollo de todo tipo de aplicaciones.

Portátil : Como el código Java se compila en bytecode, puede ejecutarse en cualquier máquina con una JVM, sin importar la arquitectura o el sistema operativo.

Rendimiento : Aunque no es tan rápido como lenguajes compilados directamente como C o C++, Java sigue teniendo un buen rendimiento gracias a optimizaciones en la JVM.
